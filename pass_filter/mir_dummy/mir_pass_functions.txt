AbortUnwindingCalls: Transforms calls that are not allowed to unwind into calls that abort if unwinding occurs. (将不允许解栈的调用转换为若发生解栈则终止程序的调用)
AddCallGuards: Adds guard code (such as abort on unwind) to function calls to ensure safety properties. (为函数调用添加保护代码如解栈时终止，以确保安全属性)
AddMovesForPackedDrops: Adds temporary moves when dropping fields of packed structs to handle alignment constraints. (在丢弃打包结构体的字段时添加临时移动，以处理对齐约束)
CleanupPostBorrowck: Removes temporary data structures and artifacts left over from the borrow checker. (移除借用检查器遗留的临时数据结构和伪像)
CopyProp: Copy Propagation. Replaces a copy's destination with its source if the source is not modified. (复写传播。如果源未被修改，则用源替换拷贝的目标)
DataflowConstProp: Constant propagation based on dataflow analysis, evaluating constant expressions at compile time. (基于数据流分析的常量传播，在编译时计算常量表达式)
DeadStoreElimination-final: Final round of Dead Store Elimination, removing stores to memory that are never read. (最后一轮死存储消除，移除从未被读取的内存存储操作)
DeadStoreElimination-initial: Initial round of Dead Store Elimination. (初始轮死存储消除)
Derefer: Simplifies dereference operations, often removing redundant `*&` sequences. (简化解引用操作，通常移除冗余的 `*&` 序列)
DestinationPropagation: Propagates the destination place of an assignment into the instruction generating the value to avoid unnecessary copies. (将赋值的目标位置传播到生成该值的指令中，以避免不必要的拷贝)
EarlyOtherwiseBranch: Optimizes `match` statements by merging branches with identical bodies or simplifying discriminant checks. (通过合并具有相同主体的分支或简化判别式检查来优化 `match` 语句)
ElaborateBoxDerefs: Lowers `box` pointer dereferences into raw pointer operations. (将 `box` 指针解引用降级为原始指针操作)
ElaborateDrops: Replaces high-level `drop` terminators with explicit drop flags and conditional drop ladders. (将高级 `drop` 终结符替换为显式的丢弃标志和条件丢弃梯)
EnumSizeOpt: Optimizes the storage layout and access of enums, potentially utilizing niches. (优化枚举的存储布局和访问，可能利用空位 niche)
EraseDerefTemps: Removes temporary local variables created for dereferences that are no longer needed. (移除不再需要的为解引用创建的临时局部变量)
ForceInline: Inlines functions explicitly marked with `#[inline(always)]`. (内联显式标记为 `#[inline(always)]` 的函数)
GVN: Global Value Numbering. Identifies and merges equivalent computations throughout the function. (全局值编号。识别并合并函数中等价的计算)
ImpossiblePredicates: Removes basic blocks that are only reachable if a predicate known to be false is true. (移除只有在已知为假的谓词为真时才能到达的基本块)
Inline: Performs function inlining based on heuristics to reduce call overhead. (基于启发式方法执行函数内联以减少调用开销)
InstSimplify-after-simplifycfg: Instruction simplification run after control flow graph simplification. (在控制流图简化之后运行的指令简化)
InstSimplify-before-inline: Instruction simplification run before inlining. (在内联之前运行的指令简化)
JumpThreading: Redirects jumps to a basic block directly to its successor if the block is empty or simple. (如果基本块为空或简单，则将跳转重定向到该块的后继)
LowerIntrinsics: Lowers generic Rust intrinsics (like `std::mem::size_of`) into MIR instructions or LLVM intrinsics. (将通用 Rust 内建函数如 `std::mem::size_of` 降级为 MIR 指令或 LLVM 内建函数)
LowerSliceLenCalls: Replaces calls to `slice.len()` with direct access to the slice's length field. (将 `slice.len()` 调用替换为直接访问切片的长度字段)
MatchBranchSimplification: Simplifies complex control flow structures generated by `match` expressions. (简化由 `match` 表达式生成的复杂控制流结构)
PostAnalysisNormalize: Normalizes MIR after various analysis passes to ensure consistent form. (在各种分析 pass 之后标准化 MIR 以确保形式一致)
PreCodegen: Final cleanup and preparation pass before generating LLVM IR. (生成 LLVM IR 之前的最终清理和准备 pass)
PromoteTemps: Promotes temporary variables to static constants if they qualify (e.g., references to constants). (如果符合条件例如对常量的引用，将临时变量提升为静态常量)
ReferencePropagation: Propagates references to avoid unnecessary indirections and copies. (传播引用以避免不必要的间接寻址和拷贝)
RemoveNoopLandingPads: Removes landing pads (cleanup blocks) that perform no actions other than resuming unwinding. (移除除了恢复解栈外不执行任何操作的落地垫即清理块)
RemovePlaceMention: Removes `PlaceMention` statements which are used for debug info or analysis but not needed for execution. (移除 `PlaceMention` 语句，这些语句用于调试信息或分析，但执行不需要)
RemoveUnneededDrops: Removes drop terminators for types that don't need dropping (Copy types) or are already moved. (移除不需要丢弃的类型如 Copy 类型或已移动值的 drop 终结符)
RemoveZsts: Removes fields and locals of Zero-Sized Types since they occupy no memory. (移除零大小类型的字段和局部变量，因为它们不占用内存)
ScalarReplacementOfAggregates: Replaces aggregate types (structs, tuples) with individual local variables for their fields. (将聚合类型结构体、元组替换为其字段的单独局部变量)
SimplifyCfg-after-unreachable-enum-branching: Control Flow Graph simplification run after removing unreachable enum branches. (在移除不可达枚举分支后运行的控制流图简化)
SimplifyCfg-final: Final round of Control Flow Graph simplification. (最后一轮控制流图简化)
SimplifyCfg-initial: Initial round of Control Flow Graph simplification. (初始轮控制流图简化)
SimplifyCfg-post-analysis: Control Flow Graph simplification run after analysis. (分析之后运行的控制流图简化)
SimplifyCfg-pre-optimizations: Control Flow Graph simplification run before main optimizations. (主优化之前运行的控制流图简化)
SimplifyCfg-promote-consts: Control Flow Graph simplification run during constant promotion. (常量提升期间运行的控制流图简化)
SimplifyComparisonIntegral: Simplifies integer comparison operations. (简化整数比较操作)
SimplifyConstCondition-after-const-prop: Simplifies constant conditions after constant propagation. (在常量传播之后简化常量条件)
SimplifyConstCondition-after-inst-simplify: Simplifies constant conditions after instruction simplification. (在指令简化之后简化常量条件)
SimplifyConstCondition-final: Final round of constant condition simplification. (最后一轮常量条件简化)
SimplifyLocals-after-value-numbering: Removes unused local variables after value numbering. (在值编号之后移除未使用的局部变量)
SimplifyLocals-before-const-prop: Removes unused local variables before constant propagation. (在常量传播之前移除未使用的局部变量)
SimplifyLocals-final: Final round of removing unused local variables. (最后一轮移除未使用的局部变量)
SingleUseConsts: Optimizes handling of constants that are used only once. (优化仅使用一次的常量的处理)
StateTransform: Transforms the state machine of generators/async functions. (转换生成器/异步函数的状态机)
Subtyper: Handles subtyping relationships and variance analysis (often part of NLL/borrowck). (处理子类型关系和变变性分析，通常是 NLL/借用检查的一部分)
UnreachableEnumBranching: Removes branches matching enum variants that cannot exist (e.g., uninhabited types). (移除匹配不存在枚举变体如无人居住类型的分支)
UnreachablePropagation: Propagates unreachable status through the control flow graph to eliminate dead blocks. (在控制流图中传播不可达状态以消除死块)
analysis: Snapshot of MIR after initial analysis phases (borrow checking, etc.). (初始分析阶段如借用检查之后的 MIR 快照)
analysis-post-cleanup: Snapshot of MIR after cleanup following analysis. (分析后清理之后的 MIR 快照)
built: The initial MIR generated directly from HIR (High-Level IR). (直接从 HIR 高级 IR 生成的初始 MIR)
runtime: MIR version used for code generation (monomorphized). (用于代码生成的 MIR 版本单态化)
runtime-optimized: Optimized MIR version used for code generation. (用于代码生成的优化后 MIR 版本)
runtime-post-cleanup: Snapshot of MIR after final cleanup before codegen. (代码生成前最终清理之后的 MIR 快照)
